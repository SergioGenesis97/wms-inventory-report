# WMS Inventory Report

## Descripción
Sistema en Python que genera un reporte de inventario clasificando productos en:
- Reabastecer
- Stock bajo
- Stock suficiente

## Cómo ejecutar

1. Clona el repositorio
2. Ejecuta el archivo principal:

python execute.py

## Tecnologías
- Python 3
- Git
- GitHub
